# Multi-Page E-Learning Platform

This project is a frontend-based multi-page web application developed using HTML, CSS, Bootstrap, and JavaScript.  
It simulates an online learning platform with courses, quizzes, and progress tracking without using a backend server.

## Project Overview
The application allows users to:
- Navigate through different pages
- View available courses
- Attempt quizzes
- Track learning progress
- Manage their profile

## Pages
- Dashboard.html – Displays overview and progress
- Courses.html – Shows course list and lessons
- Quiz.html – Dynamic quiz generation
- Profile.html – User details and completed courses

## Features
- Multi-page navigation with breadcrumb
- Responsive design using CSS Grid & Flexbox
- Dynamic quiz generation using JavaScript
- Asynchronous data loading simulation (Promise, async/await)
- Local Storage for saving progress and results
- Interactive UI with events and DOM manipulation
- Progress tracking using <progress> element

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## Testing
Basic Jest test cases implemented for:
- Grade calculation
- Score percentage
- Pass/Fail logic

## How to Run
1. Download or clone the repository
2. Open Dashboard.html in any browser
